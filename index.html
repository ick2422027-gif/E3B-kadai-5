ドットマトリクスLEDによる安全・危険の視覚化表示

LED班：藤原海音 / 鈴木敬吾

7セグメントLEDでの数値出力実験を経て、発光数をさらに増やしたドットマトリクスLEDを用いた絵文字（笑顔など）の出力に挑戦しました。熱中症の危険度を視覚的に判別できるようにするための基礎研究です。プログラム内でビットパターンデータを定義し、右ビットシフト演算（>>）と論理積（& 0x01）を組み合わせた型抜き処理（ビット判定ロジック）を実装。8マスのうち光らせたいマスの情報だけを一瞬の計算でピンポイントに抽出・制御することに成功しました。

7セグメントLEDを用いた数値出力と共通設計図の構築

LED班：藤原海音 / 鈴木敬吾

バーサライタでの将来的な文字出力を見据え、目視で情報がはっきり分かるように7セグメントLEDを用いた0〜9のカウントアップ回路を設計しました。前回のdelay()関数によるミリ秒単位の速度制御を応用したほか、使用する7セグの種類（カソードコモン／アノードコモン）の性質の違いに対応するため、プログラム側で自動的にHIGHとLOWの信号を反転させて翻訳する「共通設計図（自動判定ロジック if (!IS_CATHODE_COMMON)）」を組み込み、部品が変わってもそのまま動く柔軟なコードを開発しました。

Arduinoによる3色LEDの条件分岐ウェーブ点灯制御

LED班：藤原海音 / 鈴木敬吾

3色同時点灯の成功を踏まえ、Arduino IDEを用いた本格的なプログラム制御に移行しました。変数 count を定義して時間の経過とともにループ内で加算させ、if - else if による条件分岐（count <= 10、20、30）を構築。11番・12番・13番の各ピンへの出力を順次切り替えることで、赤・黄・緑の3色のLEDが流れるようにウェーブ状に点灯する回路とアルゴリズムを完成させました。

温度センサー（TMP36）の特性理解とシミュレーション検証

温度センサー班：鈴木暖人 / 増田淑紀

熱中症検知の頭脳となる温度センサー（TMP36）のデータシートを読み解き、温度変化に伴う内部抵抗の変化（0℃の時に500mV出力、1℃上昇ごとに10mV増加する特性）について学びました。その後、Tinkercadを用いたオンラインシミュレーション環境を構築し、センサーとマルチメーターを接続。任意の温度設定における出力電圧を測定し、メーカーが提供するデータシートの理想特性曲線と一致するかを比較・検証しました。

LEDの前提知識習得と3色同時点灯の基礎実験

LED班：藤原海音 / 鈴木敬吾

電子工作の第一歩として、LEDの極性（足の長いアノードがプラス、短いカソードがマイナス）や、データシートで定格電流（DC Forward Current：一般的には20mA）を確認する重要性を学習しました。その後、Arduino Uno、ブレッドボード、330Ωの抵抗3個を使用し、赤・黄・緑の3つのLEDを過電流なく安全に同時点灯させる基本回路をブレッドボード上に構築しました。
